## Settingup version control system in Ubuntu
- RabbitVCS
  ```
   sudo apt update
  ```
  ``` 
  sudo apt install rabbitvcs-core rabbitvcs-cli rabbitvcs-nautilus
  ```
- Commandline only Subversion
  ```
  sudo apt-get install subversion
  ```
